 🏥 Smart Hospital Decision Support System

An AI-powered healthcare decision support system that assists healthcare professionals in predicting patient outcomes using Machine Learning. The system helps improve clinical decision-making by predicting diabetic readmission, hospital length of stay, pneumonia diagnosis, and patient health outcomes.

---

 📌 Project Overview

The Smart Hospital Decision Support System leverages Machine Learning models to analyze patient data and provide predictive insights for hospitals. It supports doctors, nurses, and hospital administrators in making faster, data-driven decisions to improve patient care and optimize hospital resources.

 🚀 Features

- 🩺 Diabetic Readmission Prediction
- 🏥 Length of Stay Prediction
